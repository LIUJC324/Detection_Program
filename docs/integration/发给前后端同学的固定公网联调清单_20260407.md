实时流启动请求示例
POST /v1/inference/stream/start
{
  "sessionId": "stream_demo_001",
  "rgbPullUrl": "http://8.137.107.232:8088/live/001_rgb.flv",
  "irPullUrl": "http://8.137.107.232:8088/live/001_ir.flv",
  "resultCallbackUrl": "http://8.137.107.232:8080/api/v1/detect/model/result",

  "rgbPosition": "left",
  "sampleFps": 4,
  "callbackToken": "change-me"
}

说明：

1. `sampleFps` 现在保留，建议继续传。
2. 不传时模型端走部署默认值。
3. 当前默认值来自部署配置 `default_stream_sample_fps`，稳定部署通常为 `2.0`。
4. 可选传 `callbackMinIntervalMs` 覆盖模型端默认回调节流；传 `0` 表示不额外限速。
5. 除 `sourceUrl -> rgbPullUrl + irPullUrl` 外，其余字段口径保持不变，仍可继续传。
