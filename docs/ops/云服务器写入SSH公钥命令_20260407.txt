云服务器：将本机公钥安全写入 /root/.ssh/authorized_keys

使用说明：
1. 在阿里云 Linux 服务器终端中直接整块复制执行下面命令。
2. 这套写法不会因为终端显示自动换行而把公钥写坏。
3. 执行完成后，终端会输出：
   - OK
   - 最后一行完整公钥

如果服务器里没有 python3，先执行：

apt update && apt install -y python3

执行完后可再确认：

tail -n 1 /root/.ssh/authorized_keys

如果输出的最后一行以：

ssh-rsa AAAAB3NzaC1yc2EAAAADAQABAAACAQ...

开头，并以：

liujuncheng@rog14air

结尾，就说明公钥已正确写入。
